This is a basic vision project directory that contains:

- various image reading routines
- a basic pixel representation data structure
- a demo version of something close to a Harris corner detector
- a file format for corner points that canbe read by mutlab
- a file format for writing out images of integer labels as a run length encoded map.

- some mutlab/octave bits for displaying run length map image output.
